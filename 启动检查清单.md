## 🚀 启动步骤

### 1. 配置 Anthropic API Key（必须）

编辑 `.env` 文件，找到这一行：
```bash
ANTHROPIC_API_KEY=your_anthropic_api_key_here
```

替换为你的真实 API Key：
```bash
ANTHROPIC_API_KEY=sk-ant-api03-xxxxx
```

python run.py

## ⚠️ 如果启动失败

### 错误1：AttributeError: 'function' object has no attribute 'mkdir'
- 原因：配置属性缺少 @property 装饰器
- 解决：已修复，重新启动即可

### 错误2：ModuleNotFoundError
- 原因：缺少依赖
- 解决：`pip install -r requirements.txt`

### 错误3：Anthropic API 错误
- 原因：API Key 未配置或无效
- 解决：检查 `.env` 中的 `ANTHROPIC_API_KEY`
