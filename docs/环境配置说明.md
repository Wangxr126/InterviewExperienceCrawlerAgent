## 📝 其他项目如何配置？

### 模板：ENVIRONMENT.md

```markdown
# 项目环境配置

## Conda环境
- 环境名称：`你的环境名`
- Python版本：3.x

## 激活命令
\`\`\`bash
conda activate 你的环境名
\`\`\`

## 运行命令
\`\`\`bash
conda activate 你的环境名
python run.py
\`\`\`
```

### 模板：start.bat

```batch
@echo off
call conda activate 你的环境名
python run.py
pause
```
