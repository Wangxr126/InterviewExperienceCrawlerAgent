   ↓
5. 设置保护标志（立即生效）
   ┌─────────────────────────────────┐

   └─────────────────────────────────┘
   ↓

   │ while (true) {                  │

   ┌─────────────────────────────────┐

   └─────────────────────────────────┘

## 关键技术点

### 1. 为什么需要 sendInProgress？

```javascript
// ❌ 只用 loading.value（响应式变量）
if (!text || loading.value) {
  return
}
loading.value = true  // 更新可能有延迟

// ✅ 双重检查（响应式 + 普通变量）
if (!text || loading.value || sendInProgress) {
  return
}
sendInProgress = true  // 立即生效
loading.value = true   // 响应式更新
```

### 2. 为什么需要 finally 块？

```javascript
try {
  // 发送请求
} catch (err) {
  // 处理错误
} finally {
  // ✅ 无论成功还是失败，都要重置标志
  loading.value = false
  sendInProgress = false
}
```

### 3. 为什么需要按钮禁用？

```vue
<!-- ✅ 视觉反馈 + 功能保护 -->
<el-button 
  :disabled="loading || sendInProgress"
  @click="send">
  发送
</el-button>
```

## 测试验证点

```

```

### ✅ 快速点击
```
点击1 → 发送
点击2 → 被阻止（sendInProgress=true）

```

### ✅ 错误处理
```
发送失败 → finally 块执行 → 标志重置 → 可以重试
```

```
发送中 → 点击快捷按钮 → 显示提示 → 操作被阻止
```
