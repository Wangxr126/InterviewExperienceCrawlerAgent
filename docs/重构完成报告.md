```
┌─────────────────────────────────────────┐
│         爬虫层（Hunter Service）         │

│  LLM：❌                                 │

              ↓

              ↓

└─────────────────────────────────────────┘
```

## 只有 2 个真正的 Agent

1. **Miner Agent**（信息挖掘师）
   - 继承 `ReActAgent`
   - 使用 LLM 进行语义理解
   - 从面经原文中挖掘结构化信息

2. **Interviewer Agent**（面试官）
   - 继承 `ReActAgent`
   - 使用 LLM 进行对话
   - 面试、出题、评估
