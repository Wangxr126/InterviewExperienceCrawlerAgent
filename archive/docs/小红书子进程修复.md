# Playwright 错误快速修复

## 🐛 错误
```
NotImplementedError
at _make_subprocess_transport
```

## 🔍 原因
Windows 的 `SelectorEventLoop` 不支持子进程，而 Playwright 需要启动浏览器子进程。

## ✅ 修复

**文件**：`backend/services/crawler/xhs_crawler.py` (第 506 行)

**修改**：
```python
# 修改前
asyncio.set_event_loop_policy(asyncio.WindowsSelectorEventLoopPolicy())

# 修改后
asyncio.set_event_loop_policy(asyncio.WindowsProactorEventLoopPolicy())
```

## 📊 对比

| 事件循环 | 支持子进程 | 支持 Playwright |
|---------|-----------|----------------|
| SelectorEventLoop | ❌ | ❌ |
| ProactorEventLoop | ✅ | ✅ |
