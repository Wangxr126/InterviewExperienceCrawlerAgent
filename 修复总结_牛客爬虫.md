# 牛客爬虫修复总结

## 修复时间：2025-03-08

## 问题列表及解决方案

### 1. ✅ 多页爬取失败
**问题**：第2页及以后的页面无法爬取，返回编码错误
**原因**：第1页使用HTML解析，第2+页使用POST API，但实现不一致
**解决**：统一所有页面使用POST API接口（参考test4.py的正确实现）

### 2. ✅ 中文编码错误
**问题**：`'latin-1' codec can't encode characters in position 47-48`
**原因**：HTTP headers中的Referer包含中文关键词，无法用latin-1编码
**解决**：使用 `urllib.parse.quote()` 对中文关键词进行URL编码

### 3. ✅ 图片提取函数缺失
**问题**：`name '_collect_image_urls' is not defined`
**原因**：函数被调用但没有定义
**解决**：添加 `_collect_image_urls()` 函数，整合DOM和JSON两种提取方式

### 4. ✅ 时区问题
**问题**：系统时间不是北京时间
**原因**：使用系统本地时间，没有明确指定时区
**解决**：
- 创建统一的时间工具模块 `backend/utils/time_utils.py`
- 全局替换所有时间相关代码，统一使用北京时间 (UTC+8)
- 修改了8个文件的时间处理逻辑

```python
def _collect_image_urls(soup: BeautifulSoup, html: str) -> List[str]:
    # 优先使用JSON方式提取图片
    images = _collect_image_urls_from_json(html)
    
    # 如果JSON方式没有提取到图片，尝试DOM方式

        images = _collect_image_urls_from_dom(soup)
    
    return images
```
