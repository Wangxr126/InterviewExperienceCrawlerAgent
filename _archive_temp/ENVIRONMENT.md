# 项目环境配置

## Conda环境
- 环境名称：`NewCoderAgent`
- Python版本：3.x

## 激活命令
```bash
conda activate NewCoderAgent
```

## 运行命令
```bash
# 启动后端
conda activate NewCoderAgent
python run.py

# 运行测试
conda activate NewCoderAgent
pytest

# 其他Python命令
conda activate NewCoderAgent
python xxx.py
```

## 注意事项
- 所有Python相关命令都需要先激活此环境
- 如果环境未激活，可能会导致依赖包找不到
