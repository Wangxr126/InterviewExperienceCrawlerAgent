## 📝 维护建议

### 日常开发

1. **新建临时脚本**
   - 放在 `scripts/temp/` 目录
   - 命名规范：`功能_日期.py`

2. **新建临时文档**
   - 放在 `docs/archive/` 目录
   - 命名规范：`主题_日期.md`

3. **定期清理**
   - 每周清理一次临时文件
   - 重要的脚本移到 `backend/scripts/`
   - 重要的文档移到 `docs/`

### Git提交

1. **提交前检查**
   ```bash
   git status
   # 确保不包含：
   # - .claude/
   # - docs/archive/
   # - scripts/temp/
   # - .env
   ```

2. **提交规范**
   ```bash
   git add .
   git commit -m "feat: 功能描述"
   git push
   ```
